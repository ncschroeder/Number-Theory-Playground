Number Theory Playground

Home

Number theory is a branch of math and is the study of integers. Let's go over a few types of numbers. Natural numbers are 1, 2, 3, and so on. Whole numbers are the natural numbers along with 0. Integers are the whole numbers along with negative natural numbers; so ..., -2, -1, 0, 1, 2, ...

The Number Theory Playground is an application where a user can visit sections by clicking the buttons for them at the top. These sections show info and do calculations for number theory concepts. These calculations are done based on input numbers provided by the user. Some sections require 1 input number and some require 2. An example of a section is the prime numbers section. It has 17 sentences of info and can find the first 30 prime numbers that are ≥ an input number. Plenty of info about the sections can be found by visiting them and clicking "Info" or the triangle to the left of that to expand the collapsible info section.

Almost all of the calculations involve only natural numbers. There are 2 exceptions to this:

  1. The divisibility section does calculations that might involve negative integers or 0.
  2. The Fibonacci-like sequences section does calculations where the result is a floating-point number.

The term "whole number" is often used in this app since I find it to be more self-explanatory than "natural number."

More info about this website and project can be found in the GitHub repo for it.
